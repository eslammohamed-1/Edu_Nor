# EduNor — Social Media Brand Guide | دليل هوية السوشيال ميديا

---

## 📌 الهدف من الدليل ده

الدليل ده بيضمن إن أي صورة أو بوست أو فيديو يتعمل لـ EduNor — سواء بإيدك أو بالذكاء الاصطناعي — يبقى **شكله واحد** وهوية المنصة واضحة على كل المنصات.

## 🎯 القواعد العامة (تنطبق على كل المنصات)

### الألوان الأساسية — دايمًا موجودة

| اللون | الكود | الاستخدام في السوشيال |
|-------|-------|----------------------|
| Navy Blue | `#1E3A5F` | الخلفيات الأساسية، النصوص |
| Golden Yellow | `#F4A825` | العناوين المميزة، الـ CTA، الهايلايت |
| White | `#FFFFFF` | النصوص على خلفية غامقة، فراغات |
| Soft Teal | `#2EC4B6` | عناصر ثانوية، badges، أرقام |
| Dark Gray | `#333333` | نصوص على خلفية فاتحة |
| Light Gray | `#F5F5F5` | خلفيات فاتحة بديلة |

### نسب استخدام الألوان في أي تصميم

```
Navy Blue:     60%  ████████████████████░░░░░░░░░░░░  (الخلفية الأساسية)
Golden Yellow: 25%  ████████░░░░░░░░░░░░░░░░░░░░░░░░  (العناوين + CTA)
White/Teal:    15%  █████░░░░░░░░░░░░░░░░░░░░░░░░░░░  (تفاصيل + أيقونات)
```

> [!IMPORTANT]
> الخلفية **دايمًا** Navy Blue أو White — مفيش ألوان تانية كخلفية رئيسية.

### الخطوط

| الاستخدام | الخط | الحجم النسبي |
|-----------|------|-------------|
| العنوان الرئيسي (عربي) | **Cairo Bold** | كبير جدًا — يملا 40-60% من التصميم |
| العنوان (إنجليزي) | **Poppins Bold** | كبير جدًا |
| النص الثانوي | Cairo Regular / Poppins Regular | متوسط |
| الـ CTA | Cairo Bold | متوسط-كبير |

### عناصر بصرية ثابتة

| العنصر | الوصف |
|--------|-------|
| **اللوجو** | دايمًا موجود — الزاوية العلوية اليمنى أو السفلية |
| **أيقونات تعليمية** | كتاب مفتوح 📖، لمبة 💡، قبعة تخرج 🎓، جراف صاعد 📈 |
| **أشكال هندسية** | دوائر ناعمة، خطوط منحنية — مفيش زوايا حادة |
| **الـ CTA** | زرار ذهبي بـ rounded corners دايمًا |

## 📂 أنواع البوستات وقوالبها

### 1. 📚 بوست درس جديد (Lesson Announcement)

**التصميم:**
- خلفية: Navy Blue solid أو Navy gradient
- عنوان الدرس: كبير بالذهبي في النص
- أيقونة المادة: في الزاوية
- اسم المادة + الصف: نص أبيض تحت العنوان
- CTA: زرار ذهبي "شوف الدرس"
- اللوجو: زاوية علوية يمين

**تخطيط:**
```
┌─────────────────────────┐
│  [Logo]           [Icon]│
│                         │
│    📖 المادة: رياضيات    │
│                         │
│   ██ عنوان الدرس ██     │
│   (Golden Yellow, Big)  │
│                         │
│    الصف الثالث الإعدادي  │
│                         │
│   [═══ شوف الدرس ═══]  │
│      (Gold Button)      │
│                         │
│   EduNor • إديو نور     │
└─────────────────────────┘
```

## ✅ Do's & ❌ Don'ts

### ✅ افعل

- ✅ استخدم Navy Blue كخلفية رئيسية في أغلب البوستات
- ✅ خلي العنوان الذهبي هو أول حاجة العين تشوفها
- ✅ حط اللوجو في كل تصميم
- ✅ استخدم أيقونات flat بسيطة
- ✅ سيب مساحات فاضية (whitespace) — متزنقش التصميم
- ✅ استخدم CTA واضح في كل بوست
- ✅ خلي الخط العربي كبير وواضح
- ✅ حافظ على نفس الـ style في كل البوستات

### ❌ لا تفعل

- ❌ متستخدمش ألوان برا الباليت (لا أحمر، لا بنفسجي، لا أخضر فاتح)
- ❌ متحطش صور stock عشوائية
- ❌ متستخدمش خطوط تانية غير Cairo و Poppins
- ❌ متعملش تصميم مزدحم — قاعدة: لو شيلت عنصر ومأثرش، شيله
- ❌ متغيرش ألوان اللوجو
- ❌ متستخدمش gradients ألوان غير المعتمدة
- ❌ متخليش النص صغير ومش واضح
- ❌ متستخدمش إيموجي كتير في التصميم نفسه (الكابشن ماشي)

## 🎬 TikTok / Reels — إرشادات خاصة

### الأسلوب البصري للفيديوهات

| العنصر | التفاصيل |
|--------|---------|
| **Intro (أول ٣ ثواني)** | عنوان ذهبي كبير على خلفية Navy — لازم يشد الانتباه |
| **Lower Third** | شريط أسفل شفاف أو Navy فيه اسم الموضوع |
| **Text on Screen** | Cairo Bold، أبيض أو ذهبي، حجم كبير ومقروء على الموبايل |
| **Transitions** | Smooth fade أو slide — مش transitions مبالغ فيها |
| **End Screen** | اللوجو + "تابعنا" + CTA ذهبي |
| **Music/Mood** | إيقاع هادي وإيجابي — تعليمي مش ترفيهي |

### أنواع الفيديوهات

| النوع | المدة | الوصف |
|-------|-------|-------|
| شرح سريع | 30-60 ثانية | شرح نقطة واحدة بسيطة |
| نصيحة | 15-30 ثانية | نصيحة دراسية سريعة |
| قبل وبعد | 15-30 ثانية | قبل وبعد الفهم |
| تحدي / سؤال | 15-30 ثانية | "تعرف الإجابة؟" |
| Behind the scenes | 30-60 ثانية | كواليس تحضير الدروس |
