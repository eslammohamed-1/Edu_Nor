# هياكل الأسئلة المدعومة (Question Structures)

يوضح هذا الملف الطريقة الصحيحة لكتابة كل نوع من أنواع الأسئلة لضمان تحويلها بشكل صحيح إلى نظام Nagwa.

---

## 1. سؤال ترتيب (Ordering)
يُستخدم لترتيب العناصر في تسلسل صحيح.
```text
type: ordering
@STEM
رتّب العناصر التالية حسب الترتيب الزمني:
@CHOICES
- العنصر الأول (الأقدم)
- العنصر الثاني
- العنصر الثالث
- العنصر الرابع (الأحدث)
```

## 2. اختيار من متعدد (MCQ)
يحتوي على إجابة صحيحة واحدة فقط (تُسبق بعلامة `*`).
```text
type: mcq
@STEM
ما هو عاصمة مصر؟
@CHOICES
* القاهرة
- الإسكندرية
- الجيزة
- أسوان
```

## 3. إجابات متعددة (MRQ)
يمكن اختيار أكثر من إجابة صحيحة (تُسبق بعلامة `*`).
```text
type: mrq
@STEM
اختر المدن الساحلية مما يلي:
@CHOICES
* الإسكندرية
- القاهرة
* مطروح
- الأقصر
```

## 4. التوصيل (Matching)
يُستخدم لتوصيل أزواج من العناصر (يُفصل بين الزوج بعلامة `|`).
```text
type: matching
@STEM
صل كل دولة بعاصمتها:
@MATCHING_PAIRS
مصر | القاهرة
فرنسا | باريس
اليابان | طوكيو
السودان | الخرطوم
```

## 5. إكمال الفراغات (Gap Fill)
يُستخدم لملء الكلمات الناقصة (الكلمات الصحيحة تُسبق بعلامة `*`).
يتم وضع `@BLANK` في المكان الذي سيظهر فيه الفراغ.
```text
type: gap
@STEM
تعتبر @BLANK عاصمة مصر، بينما تعتبر @BLANK عاصمة فرنسا.
@GAPS
* القاهرة
- الجيزة
* باريس
- لندن
```

## 6. إجابة قصيرة (String)
سؤال يتطلب كتابة إجابة نصية قصيرة ومحددة.
```text
type: string
@STEM
ما هو اسم الكوكب الاحمر؟
@ANSWER
المريخ
```

## 7. إدخال رقمي (Numeric Input)
يُستخدم للإجابات الرياضية أو الرقمية، ويمكن إضافة وحدة القياس بعد علامة `|`.
```text
type: input
@STEM
احسب: 5 + 10
@ANSWER
15 | سم
```

## 8. إجابة حرة / مقال (FRQ)
يُستخدم للأسئلة المقالية التي تتطلب كتابة نص طويل وتصحيح ذكي للغات.
```text
type: frq
@STEM
اكتب مقالاً قصيراً عن أهمية القراءة.
@ANSWER
تعتبر القراءة غذاء العقل حيث تساهم في توسيع المدارك...
```

## 9. سؤال العد (Counting)
يُستخدم في أسئلة عد العناصر في شبكة.
```text
type: counting
@STEM
كم عدد البرتقالات في الصورة؟
@ANSWER
5 | برتقالات
```

## 10. لغز الصور (Puzzle)
يُستخدم لترتيب قطع الصور لتكوين شكل كامل.
```text
type: puzzle
@STEM
رتب قطع الصورة لتكوين الشكل الصحيح.
@PIECES
Piece 1: image_url_1.png
Piece 2: image_url_2.png
Piece 3: image_url_3.png
Piece 4: image_url_4.png
@SOLUTION
2, 4, 1, 3
@COMPLETE_IMAGE
complete_image_url.png
```

## 11. سؤال استطلاع رأي (Opinion)
يُستخدم لجمع الآراء ولا توجد فيه إجابة صحيحة أو خاطئة. يتم سرد الاختيارات فقط.
```text
type: opinion
@STEM
ما هو تقييمك لمستوى صعوبة هذا الدرس؟
@CHOICES
- سهل جداً
- سهل
- متوسط
- صعب
- صعب جداً
```

## 12. سؤال مجموعات الاختيارات (GMRQ)
يُستخدم لاختيار إجابة واحدة من مجموعتين مختلفتين (مثل اختيار الفعل ثم الزمن المناسب).
```text
type: gmrq
@STEM
اختر الإجابات الصحيحة للجملة التالية:
@CHOICES
* Group A - Choice 1
- Group A - Choice 2
---
- Group B - Choice 1
* Group B - Choice 2
```

## 13. السؤال المتعدد الأجزاء (Multipart)
يُستخدم لضم عدة أسئلة تحت نص واحد (Statement). يحتوي على الجزء العام ثم الأجزاء المنفصلة.
```text
type: multipart
@STATEMENT
اقرأ النص التالي ثم أجب عن الأسئلة:
(هنا يوضع النص الطويل أو القطعة)

@PART
type: mcq
@STEM
السؤال الأول المتعلق بالقطعة؟
@CHOICES
* الإجابة الصحيحة
- إجابة خاطئة

@PART
type: string
@STEM
السؤال الثاني المتعلق بالقطعة؟
@ANSWER
الإجابة النموذجية
```

---

### ملاحظات هامة:
1. **الميتا داتا (Metadata):** يمكنك إضافة معلومات في بداية أي سؤال مثل `id: 1001` أو `language: ar`.
2. **الرموز الرياضية:** يمكنك استخدام الصيغ الرياضية بين علامات الدولار `$x^2$` أو علامات ` ``x^2`` `.
3. **الصور:** يمكن وضع رابط الصورة مباشرة في أي مكان داخل الـ `@STEM` أو `@CHOICES`.
