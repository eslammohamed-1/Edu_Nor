### Icon Style
- **Style:** Flat or Soft 3D Minimal
- **Stroke weight:** 1.5px–2px
- **Size grid:** `16px`, `20px`, `24px`, `32px`, `48px`
- **Color:** Use brand colors only — Navy, Gold, Teal, or Gray

### Illustration Style
- **Type:** Semi-flat with subtle shadows and depth
- **Characters:** Friendly, diverse students (kids & teenagers)
- **Colors:** Stick to brand palette — Navy, Gold, Teal, White
- **Backgrounds:** Clean, simple, with soft geometric shapes
- **Mood:** Warm, encouraging, approachable
- **Avoid:** Overly complex, realistic, or corporate-looking illustrations

## 🎬 Animation & Motion

### Transition Speeds

| Name | Duration | Usage |
|------|----------|-------|
| Instant | `100ms` | Button active states, toggles |
| Fast | `200ms` | Hover effects, small UI changes |
| Normal | `300ms` | Card animations, modals opening |
| Slow | `500ms` | Page transitions, large elements |

### Easing Curves

| Name | Value | Usage |
|------|-------|-------|
| Default | `ease` | General transitions |
| Smooth | `cubic-bezier(0.4, 0, 0.2, 1)` | Smooth enter/exit |
| Bounce | `cubic-bezier(0.34, 1.56, 0.64, 1)` | Playful micro-interactions |
| Sharp | `cubic-bezier(0.4, 0, 0.6, 1)` | Quick snapping movements |

### Micro-Interactions

| Element | Animation |
|---------|-----------|
| Button hover | Scale up `1.02`, add shadow |
| Button click | Scale down `0.98` |
| Card hover | Lift up `translateY(-2px)`, shadow grows |
| Page load | Fade in + slide up `(opacity 0→1, translateY(20px→0))` |
| Toast notification | Slide in from top + fade in |
| Progress bar | Width animates with `ease` over `800ms` |
| Skeleton loading | Pulse shimmer effect `(opacity 0.6→1 loop)` |

> [!NOTE]
> **Respect `prefers-reduced-motion`** — Disable non-essential animations when the user has this OS setting enabled.

## ♿ Accessibility

### Contrast Ratios (WCAG AA)

| Element | Requirement | Status |
|---------|-------------|--------|
| Body text (`#333`) on White (`#FFF`) | ≥ 4.5:1 | ✅ 12.63:1 |
| Navy (`#1E3A5F`) on White (`#FFF`) | ≥ 4.5:1 | ✅ 9.76:1 |
| White text on Navy (`#1E3A5F`) | ≥ 4.5:1 | ✅ 9.76:1 |
| Navy text (`#1E3A5F`) on Gold (`#F4A825`) | ≥ 3:1 (large text) | ✅ 3.48:1 |
| Dark text on Gold buttons | Use `#1E3A5F` not `#FFFFFF` | ✅ |

### General Rules
- **Minimum touch target:** `44px × 44px` on mobile
- **Focus indicators:** Visible `2px` outline using `#F4A825` with `3px` offset
- **Alt text:** All images must have descriptive alt text (Arabic preferred)
- **Keyboard navigation:** All interactive elements must be keyboard accessible
- **Font size minimum:** `12px` — never smaller
- **Form labels:** Every input must have a visible associated label
- **Error messages:** Must be descriptive and suggest correction

## 🔄 Loading & Empty States

### Loading States
- **Skeleton screens** preferred over spinners for content areas
- **Spinner** used only for actions (button loading, form submitting)
- Skeleton color: `#E0E0E0` pulsing to `#F5F5F5` (light mode)
- Skeleton color: `#2D3748` pulsing to `#1A2332` (dark mode)

### Empty States
- Include a **friendly illustration** matching brand style
- Add a **clear message** explaining what's missing
- Provide a **CTA button** to guide the user (e.g., "ابدأ أول درس")
- Tone: Encouraging, not blaming

### Error States
- Use **Error Red** (`#E74C3C`) for borders and icons
- Provide a **clear error message** in simple Arabic
- Include a **retry action** when possible
- Never show raw error codes to students

6. Use **defined animation curves** — no custom timings
